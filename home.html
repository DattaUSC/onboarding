About Us

Code Alliance is an open source technology for good initiative started by Benetech Labs. Our goal is to bring together the open source software, impact, and high-tech volunteerism communities to build a coalition of support for nonprofit organizations.

We do this by contacting nonprofits that are highly technical, and finding specific tasks and projects in a wide variety of fields that volunteers can hack on individually and as part of larger teams.

We specialize in open source, social impact technology, and connecting diverse stakeholders in an effort to promote scalable solutions.

What is open source?

Web Development lives in open source. Open source software can be changed and used freely by anyone, and has a world of ongoing free support through the vibrant communities surrounding each piece of software. Open source software is much less resource-intensive, and gets updated by the community itself, and so you're not required to buy new versions of the software every year, like you must with Microsoft Office. Open source software allows people to build powerful web companies and applications at low cost, and today, is responsible for thousands of startups and the millions of individuals that use them. Most of the web is built on top of open source technology.

What is GitHub?

GitHub is a free open source website that allows anyone to upload, store, edit, and track code changes by placing code files into a public repository.

The GitHub website is based on a technology called git, which is a version control system. Git is used on your local machine and allows programmers to track code changes they’ve made to specific applications - thereby allowing them to go back to previous branches of the code, or create new branches without erasing the initial or previous application. This allows any programmer with access to that repository to manipulate and see the various branches that are being created.

Setup instructions

Setup Steps:

  1. Go to www.github.com
  2. Sign-up for a Username and Password
  3. Choose the “Free Plan” and verify your account through email if asked.
  4. Go to https://desktop.github.com/ and Download GitHub For Desktop (make sure to download the Mac or Windows version based on your machine).
  5. Now, open GitHub For Desktop and login using your GitHub credentials
  6. Now, go to your GitHub.com account on the webpage, in the top-right hand corner you’ll see an arrow, on the drop-down menu, click “your profile”.
  7. At this point, you’re all setup!

Steps to link to another account

  1. Search for “User” and on the left side, click on Users, and you’ll see a list of users, click the name that you’re looking for.
  2. Go to the top-right and click on “Fork” and this clones that repository to your account. Then, in your cloned copy, look for the icon - monitor with down arrow. Click on it, and that will download this to your local machine and store it in GitHub For Desktop. Now, you have a local copy of the repository.

How to push changes to the master branch

In the GitHub for Desktop tool, you can follow these steps to commit your changes

  • Commit with note
  • Push
  • Sync

This synchronizes your local changes to the repository on GitHub that you own. Once you are happy with the changes you’ve made to your repository, you push them back to the master branch (where you cloned the original repository)
